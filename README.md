# Hero Combat Sandobx
## The features implemented in the project:
- Playable character with walking animation blending
- Combat animations, two attack moves: kick and punch
- The enemy npc chasing the player and attacking them
## Assets used
- RamsterZ combat animations https://www.fab.com/listings/9319491d-0e91-422c-9b67-ad4bf6c01a02
- animation sequences from Lyra Template Project
- animation sequences from Third Person Starting Project
